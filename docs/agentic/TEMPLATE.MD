# Agentic Work Item Template

Copy this file into docs/agentic/in-progress and rename it using the issue ID and slug.

Prefix guide:
- AW = Agentic Work. Use for broader engineering work items (worker/services/routes, data flow, infra, architecture, or cross-cutting tasks).
- UI = User Interface. Use for focused UX or visual behavior changes (layout, states, interactions, accessibility polish, component-level presentation).

Example filename:
- AW-017-example-work-item.md
- UI-005-example-ui-item.md

---
id: AW-000
title: Replace with short issue title
status: in-progress
source: ad-hoc
area: replace-with-area
priority: medium
depends_on: []
updated: YYYY-MM-DD
---

Tip: for UI-prefixed items, set id to UI-000 style instead of AW-000.
