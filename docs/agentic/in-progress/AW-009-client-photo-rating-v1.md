## Context
Tenants need lightweight client feedback during gallery selection workflows.

## Goal
Allow email viewers to set per-photo rating states (good, bad, unrated).

## Acceptance Criteria
- [ ] Ratings are stored per photo and per email viewer.
- [ ] Latest vote replaces previous vote for same viewer/photo.
- [ ] Ratings are visible to tenant admins in management views.
- [ ] Aggregate summaries include good, bad, unrated, and disagreement.

## Notes
Phase one excludes free-form comments.
